DME PI

Welcome to the DME Python Interface. This is a web based graphical user interface in which you can program anything you want. It uses blocks, which can be added from the bottom of your screen, and once the GENERATE button is pressed, you will be presented with a python(.py) file. This can be run from the python console and edited from python IDLE which can be downloaded here.
